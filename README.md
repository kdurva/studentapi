# Student API

A RESTful API built using Spring Boot, Spring Data JPA, and H2 Database for managing student records.

## Features

* Create Student (POST)
* Get All Students (GET)
* Get Student By ID (GET)
* Update Student (PUT)
* Delete Student (DELETE)

## Technologies Used

* Java 21
* Spring Boot
* Spring Web
* Spring Data JPA
* H2 Database
* Maven
* Postman

## API Endpoints

### Create Student

POST /students

### Get All Students

GET /students

### Get Student By ID

GET /students/{id}

### Update Student

PUT /students/{id}

### Delete Student

DELETE /students/{id}
